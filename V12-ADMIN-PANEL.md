# ÉpítésNapló AI PRO v12 – Admin panel

## Új funkciók

- Új oldal: `admin-panel.html`
- Felhasználók listája
- Csomag kézi módosítása: Próba / Starter / Pro / Business / Lejárt
- AI kredit kézi jóváírása
- Admin jog adása / elvétele
- Felhasználó tesztnézet megnyitása
- KPI összesítő: felhasználók, Pro/Business, AI kreditek, fizetések

## Telepítés

1. Töltsd fel a ZIP tartalmát GitHubra felülírással.
2. Supabase SQL Editorban futtasd: `supabase-pro-v12-admin-panel.sql`
3. Nyisd meg: `https://epitesi-naplo.eu/admin-panel.html`
