# V133 – Ügyfélriport fotók + PDF végleges javítás

- A publikus ügyfélriportban akkor is visszatölti a fotókat, ha a mentett HTML-ből hiányzott a kép.
- A publikus PDF gomb már nem rejtett html2pdf exportból dolgozik, hanem látható nyomtatási/PDF nézetet nyit.
- Az ügyfél link létrehozása biztos fotós HTML-t ment, data-path támogatással.
- A főoldali ügyfélriport PDF gomb is a látható riportból nyit PDF/nyomtatási nézetet.
- Cache bust: script/project/view verziók V133-ra állítva.
