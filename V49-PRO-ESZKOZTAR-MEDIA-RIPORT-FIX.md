# V49 - PRO eszkoztar, media nagy nezet, riport javitas

Mit javit:
- A projektoldali PRO muveletek kartyas, mobilon is kezelheto eszkoztarat kaptak.
- A naplobejegyzeseknel a kepek es videok alatt kulon `Nagy nezet` es `Uj lapon` megnyitas van.
- A nagy media nezetben is van `Uj lapon` gomb.
- A heti/lezaro HTML riportokban a kepek kattinthatok es uj lapon nagyban nyithatok.
- Az ugyfelriport oldalon a privat Supabase videok friss linket kapnak, es a megnyitas link is erre all ra.

Ellenorzendo elesites utan:
- Projektoldal mobilon: a PRO eszkoztar kartyanak latszik, nem csuszik ossze.
- Ugyfelriport link: kep es video megnyithato nagy nezetben es uj lapon is.
